# OOP 다형성 강의 노트

## Key Claims

- 다형성은 **Operation Overriding**과 **Dynamic Binding** 두 메커니즘의 조합
- Overriding: Subclass가 Superclass와 동일한 prototype(시그니처)의 함수를 재정의
- Dynamic Binding: 런타임에 포인터가 실제 가리키는 객체 타입을 기준으로 호출 함수 결정
- C++에서 Dynamic Binding은 `virtual` 키워드로 활성화 (기본값은 Static Binding)
- Virtual Function을 통해 "같은 인터페이스, 다른 구현"이 가능해짐
