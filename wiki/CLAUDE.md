# wiki/ — 컴파일된 지식

**소유권**: AI가 작성하고 유지한다. soggyfries는 읽고 방향을 잡는다.

## 하위 폴더

| 폴더 | 용도 |
|------|------|
| `concepts/` | 개념 문서. 한 파일 = 한 개념. |
| `connections/` | 개념 간 연결, 구조적 유사성, 비교 |

## 개념 문서 형식 (concepts/)

```
# 개념명

한 줄 요약: 이 개념이 왜 존재하는가.

## 직관
## 개념 설명
## 코드 예제
## 왜 이렇게 설계됐는가
## 연결된 개념
## 참고
```

## 연결 문서 형식 (connections/)

구조적 유사성, 비교, 상위 추상화 등. 자유 형식이지만 반드시 양쪽 개념 페이지에 역링크 포함.

## AI 유지보수 규칙

- 새 소스 ingest 시: 관련 concept 페이지 업데이트 + index.md 갱신 + log.md 기록
- 모순·충돌 발견 시: 해당 페이지에 `> ⚠️ 충돌:` 블록으로 표시 후 soggyfries에게 보고
- 고아 페이지(링크 없는 페이지) 발견 시: connections/ 에 연결 문서 제안
