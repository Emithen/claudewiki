# raw/ — 원본 소스

**불변 원칙**: 이 폴더의 파일은 AI가 읽기만 하고 수정하지 않는다.

## 하위 폴더

| 폴더 | 용도 |
|------|------|
| `lectures/` | 수업 자료, 강의 노트 |
| `books/` | 책 메모, 독서 노트 |
| `articles/` | 웹 아티클, 스크랩 (Obsidian Web Clipper 활용) |
| `videos/` | 영상 노트, 강연 정리 |
| `thoughts/` | 흘러가는 생각, 아이디어, 연결의 순간 |

## 파일명 규칙

`YYYY-MM-DD_제목.md`

## Ingest 요청 방법

새 파일을 추가하고 나서: "raw/articles/2026-04-14_제목.md 인제스트해줘"
